# GitHub Push Instructions

## Issue: Authentication Required

You need to authenticate with GitHub to push your code. Here are your options:

### Option 1: Using GitHub CLI (Recommended)
```bash
# Install GitHub CLI if not installed
# On Windows/WSL: winget install --id GitHub.cli
# On macOS: brew install gh
# On Linux: See https://github.com/cli/cli/blob/trunk/docs/install_linux.md

# Authenticate
gh auth login

# Push to GitHub
git push origin main
```

### Option 2: Using Personal Access Token
```bash
# 1. Create a Personal Access Token:
#    - Go to https://github.com/settings/tokens
#    - Click "Generate new token (classic)"
#    - Select scopes: repo (full access)
#    - Copy the token

# 2. Update remote URL with token:
git remote set-url origin https://YOUR_TOKEN@github.com/tburgernyc/truecrime-videos-automate.git

# 3. Push
git push origin main
```

### Option 3: Using SSH
```bash
# 1. Generate SSH key (if you don't have one)
ssh-keygen -t ed25519 -C "your_email@example.com"

# 2. Add SSH key to GitHub:
#    - Copy your public key: cat ~/.ssh/id_ed25519.pub
#    - Go to https://github.com/settings/keys
#    - Click "New SSH key" and paste

# 3. Update remote URL to SSH:
git remote set-url origin git@github.com:tburgernyc/truecrime-videos-automate.git

# 4. Push
git push origin main
```

### Quick Fix: Use GitHub Web Interface
```bash
# If the above options don't work immediately:
# 1. Create a new repository on GitHub: https://github.com/new
# 2. Name it: truecrime-videos-automate
# 3. Follow GitHub's instructions to push existing repository
```
